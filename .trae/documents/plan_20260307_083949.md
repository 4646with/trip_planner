## 替换图片搜索服务为百度

### 方案

**使用百度图片搜索替换 Unsplash**

### 修改内容

1. **修改 `unsplash_service.py`** → 重命名为 `image_service.py`
   - 新增百度图片搜索方法
   - 保留 Unsplash 作为备用

2. **修改 `tools.py`**
   - 更新 `unsplash_image_search` 工具使用百度 API

3. **修改 `.env` 配置**
   - 添加百度图片 API Key 配置

### 需要你提供

请告诉我你申请的百度图片搜索服务的具体信息：
- API 接口地址
- 需要的参数（如 API Key、Secret Key 等）
- 返回数据格式

或者，如果高德 POI 搜索结果中包含图片，我们可以直接使用高德返回的图片，无需额外调用图片搜索 API。
