1. 并发写入 + 重试会造成数据重复（这个依然成立）
pythonattractions: Annotated[list, operator.add]
operator.add 是追加语义。如果 attraction_agent 因网络抖动触发了重试，第二次结果会直接追加到第一次后面，Planner 会拿到重复的景点数据。建议加一个按名称去重的 merge 函数替换 operator.add：
pythondef merge_unique_by_name(a: list, b: list) -> list:
    seen = {item.get("name") for item in a if isinstance(item, dict)}
    return a + [item for item in b if item.get("name") not in seen]
